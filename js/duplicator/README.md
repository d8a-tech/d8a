## GA4 duplicator (TypeScript)

This package contains a direct TypeScript port of the inline script in `dascript.js`.

### Build

- Production minified bundle:
  - `npm run build:prod`
  - Output: `../ga4-duplicator/dist/ga4-duplicator.min.js`

- Development (unminified + sourcemap):
  - `npm run build:dev`
  - Output: `../ga4-duplicator/dist/ga4-duplicator.js`
